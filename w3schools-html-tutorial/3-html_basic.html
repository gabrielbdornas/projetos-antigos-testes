Resumo

Todos os elementos HTML pode ter atributos

O atributo 'title' fornece informações adicionais ou "dicas" quando o mouse passa cima do texto

O atributo 'href' fornece endereço para links

Os atributos 'width' e 'height' definem tamanho para imagens

O atributo 'alt' fornece texto para leitores de tela

A convenção sugere utilização de letras minúsculas em atributos

A convenção sugere aspas duplas para valores de atributos
